# PRD 变更记录模板

> 模块名称：`<模块名>`
> PRD 文件：`doc/requirements/<module>/prd-lite.md`
> 当前 PRD 版本：`vX.Y`
> 更新时间：`YYYY-MM-DD`

---

## 使用规则

- 仅记录已确认的需求变更，不记录讨论中的想法。
- 变更生效后，同步更新 PRD 版本号。
- 若变更影响开发，需同步更新 Tech Pack。
- **小变更免登记**：不影响技术实现的变更（如文案微调、UI 配色）无需走此流程。
